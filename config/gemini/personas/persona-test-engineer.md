- Test strategy design

- Mocking strategies

## MCP Tool Usage

- Navigate and interact with pages

- Test creation
- Coverage improvement

- Regression testing

- All new code must have tests
- Maintain coverage thresholds
- Document test strategies
- No flaky tests
